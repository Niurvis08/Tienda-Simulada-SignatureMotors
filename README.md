Signature Motors: Plataforma de Venta de Vehículos
Este proyecto simula una plataforma de comercio electrónico (e-commerce) interactiva y sofisticada para la venta de vehículos de alta gama. El sitio está construido utilizando únicamente tecnologías frontend, enfocándose en la adaptabilidad, la accesibilidad (WCAG AA), y la interactividad dinámica mediante JavaScript puro.

Características Principales
Consumo Dinámico de Datos: Los vehículos se cargan dinámicamente desde un archivo JSON externo utilizando la API Fetch.
Gestión de Carrito: Funcionalidad completa de añadir, actualizar y calcular el total de los productos en un carrito de compras.
Diseño 100% Responsivo: Utilización de Bootstrap 5 para garantizar una visualización óptima en dispositivos móviles, tabletas y escritorios.
Simulación de Facturación: Generación de un PDF de factura detallada tras simular el proceso de pago, utilizando la librería jsPDF.
Delegación de Eventos: Uso de patrones de optimización de JavaScript para mejorar el rendimiento de la interacción con los productos.

Tecnologías Utilizadas
Tecnologías clave empleadas en la construcción de la plataforma:
HTML5 Semántico: Proporciona la estructura base accesible, utilizando etiquetas como <header>, <main>, <section>, y atributos role="..." y aria-"...".
CSS3 Puro: Se utiliza para estilos personalizados, efectos hover y la animación pulse del carrito.
Bootstrap 5 (CDN): Framework CSS principal para el desarrollo del layout responsivo, la barra de navegación (navbar), modales y componentes de interfaz de usuario (UI).
Font Awesome (CDN): Se utiliza para la iconografía del sitio, incluyendo iconos para el carrito, búsqueda, botones y el logo.
JavaScript (ES6+): Lenguaje de programación para la lógica central, incluyendo el consumo asíncrono de la API (async/await), la manipulación del DOM, el filtrado de productos y la gestión de la lógica del carrito.
jsPDF (CDN): Librería utilizada para la generación de documentos PDF (facturas) directamente en el navegador.
